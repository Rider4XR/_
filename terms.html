Rider4XR Demo Terms

This is a demo website for UI and workflow testing only.

Parcel Rule

Parcel must be under 5 KG and safely fit on bike. Cake, liquid, fragile, cash, jewellery and illegal items are not allowed.

Map / Location Note

Current location works only on HTTPS websites like GitHub Pages. Direct file opening may block location permission.

Security Note

GitHub Pages demo is not a secure backend. Real data must be protected with Firebase Auth, Firestore rules and Storage rules.
