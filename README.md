# Rider4XR 3D OTP 1234 Beep Free Map Demo

Features:
- Login / Signup with OTP
- Demo OTP: 1234
- Customer / Rider / Admin panels
- Free map
- Auto current location on HTTPS
- Manual Tap Pickup / Tap Drop fallback
- 3D animated bike UI
- Rider beep sound and vibration for new order
- Open route in Google Maps

Important:
- Current location needs HTTPS. Upload to GitHub Pages first.
- Rider beep needs one tap on Enable Beep. Mobile browser may pause sound in background. For real background alerts, production version needs Firebase push notification or native app/PWA notification.
