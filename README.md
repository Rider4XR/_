# Rider4XR GitHub Free Map Demo

This is a free static GitHub Pages demo.

## Features
- Mobile number login only
- No password
- No OTP login
- Customer panel
- Rider panel
- Admin demo panel
- Free OpenStreetMap map
- OSRM route
- Booking and parcel up to 5 KG
- Pickup OTP and Drop OTP inside website
- Rider accepts order
- Google Maps open buttons
- Basic pricing demo

## Pricing in demo
- 0 to 3 KM = ₹55 total
- Rider gets ₹40
- Above 3 KM = total KM × ₹15 + ₹10 fee
- Rider gets total KM × ₹12

## Important
This is a static demo using localStorage only. It is not secure for real business data.
Real secure version should use Firebase Auth, Firestore, Storage and rules.
