# Rider4XR Website

GitHub Pages ready responsive landing page for Rider4XR.

## Files
- `index.html`
- `style.css`
- `script.js`

## Deploy on GitHub Pages
1. Create a new repository named `rider4xr-website`.
2. Upload these files.
3. Go to Settings > Pages.
4. Select Deploy from Branch > main > root.
5. Open your GitHub Pages link.
